# Glossary

- **Policy**: Tool-agnostic constraints, prohibitions, pre/postconditions, halting rules.
- **Procedure**: Imperative steps (scripts, runbooks, CI jobs) that implement policy.
- **Trace/Attestation**: Evidence that policy was evaluated and complied with.
- **SΛCP**: ΣΛ Change Proposal (the only permitted mechanism for evolving the spec).
- **Clause ID**: Stable identifier used to bind policy ↔ execution ↔ trace.
