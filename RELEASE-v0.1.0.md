# ΣΛ Release v0.1.0

This is the initial public release of **ΣΛ (Sigma–Lambda)**.

## Included
- ΣΛ Operating Manual v1.1
- ΣΛ Change Control Protocol
- Formal governance and contribution process
- Example policy / ops / trace decomposition
- Whitepaper and positioning documents

## Scope
This release defines:
- The ΣΛ constraint language
- The procedural ingestion architecture
- The change proposal and versioning system

This release does **not** include:
- Execution tooling
- CI/CD integrations
- Policy registries as a service
- Commercial offerings

## Stability
The core specification is conservative by design.
Future releases will prioritize:
- backward compatibility
- ambiguity reduction
- explicit halting semantics

## Status
This release is intended for:
- early adopters
- infrastructure teams
- regulated environments
- governance-first AI systems
